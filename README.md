# Test Task — Meter List

Frontend приложение для отображения списка счётчиков с адресами и пагинацией.

## Stack

- React
- TypeScript
- MobX-State-Tree
- Axios
- Vite

## Features

- Получение списка счётчиков
- Получение и кэширование адресов
- Оптимизация загрузки адресов (запрашиваются только отсутствующие)
- Пагинация
- Удаление счётчиков
- Формирование полного адреса для отображения

## Setup

```bash
npm install
npm run dev
```
